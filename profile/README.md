# Nagomi Project
Nagomi is an open source, general purpose, Discord bot. Currently a work in progress.

## The Motivator
As someone who has managed servers for quite a bit, the amount of bots that are needed to manage a server is quite annoying. A lot of these Discord bots attempt to implement everything they can, but some do other features better than others.  **Nagomi is meant to change that.** Yes, this is basically another *"✨ all purpose ✨"* Discord bot.

## Planned Features
- **Activity Points and Leveling.** Nagomi is meant to strike balance between community management and fun. From my own experience, community members love this sort of feature, it entices people to talk in the server more, whether it's to get activity roles or to be at the top of the leaderboard.
- **Extensive Moderation Features.** YAGPDB has some really good moderation features, but in my personal opinion does not handle some as good as Dyno. The goal is to allow managers to have extensive control over moderation features, while also being plug-and-play for those who do not understand how to use the advanced features. An end goal to this would be to have a catalog of official and community-made filters that anyone can just add to their server.
- **Voice Rooms.** Allowing creation of a custom voice channel through another voice channel.
- **Mod Mail and Tickets System.** Allows for members to get in touch with server moderators. Some people perfer a mod mail system, some perfer tickets.
- **Web Panel for easy management.** This is just a given, but there would be a web panel that allows you to manage every portion of the bot. There will be some commands for quickly updating some things, but a majority will be done through this panel.
- .. more in the future, maybe.

## Closing
The bot will likely never have a officially hosted version due to server costs, but the goal is to have it be incredibly simple to host it yourself. I've made plenty of (horrible) Discord bots, this is really just a project for me to make a really good one that I would actually want to use myself long-term.
